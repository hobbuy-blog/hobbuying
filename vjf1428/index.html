VAIO F14 (VJF1428)を購入！ - 2026.1.26

遂に…！ちゃんと新品のパソコンを購入しました！！
ということで今回の主役はVAIO F14 VJF1428です．
詳しいレビューは後日に書きますが，ここでは概略とファーストインプレッションだけ書いておきます．
新春セールの間に買ったので，ちょっと安く手に入りました．
参考程度に，価格はソフトケース(3,000円)込みで217,800円でした．破産するって．※高校生

ちなみに構成はガッツリ盛っています．
なにせ回路SIMソフト(LTspice)とかEDAソフト(KiCAD)とか3D CADソフト(Autodesk Fusion / Shapr3D)とかを同時起動しがちなので，RAMの量が欲しいわけです．
昨今の価格高騰はありますがここだけは外せない．
しっかり32 GB積んでいます．
あとはPro版Windowsも外せません．
そんなこんなで高級構成になってしまい，以下の構成です．

  • CPU : Intel Core 7 150U
    10C12T (2P8E) | 1.8/5.4 GHz
  • RAM : LPDDR5X 32 GB
  • ROM : PCIe 4.0 (NVMe) 1 TB
  • OS : Windows 11 Pro

なかなか盛り盛り構成です．
たぶんそこらの格安ゲーミングノートPCよりかは高いと思います．
ただ，めちゃくちゃ作り込みと言うか，品質と質感が素晴らしいんですよね．

初期セットアップを済ませたら，色々なソフトをインストールします．
代表例は以下のとおりです．
  • Autodesk Fusion Personal
  • Shapr3D
  • KiCAD
  • Visual Studio Code
  • Affinity

他にもありますがこれくらいにしておきます．
かれこれ全てのインストールに5時間くらいかかりました．

ただ…これで終わりではないのです．
Windows 11のカスUIを討伐しなければなりません．
そこで登場，Explorer Patcherの登場です．
これを適用するとUIがWindows 10風のUIに変身します．
Twitter(現X)にて，まくろ@n月病(さんに教えてもらいました．(マジ感謝)
これを適用し，幸せになった後のデスクトップをお見せします．

いやー美しい．

ということでファーストインプレッションに移ります．
とりあえず本体の質感が良いです．
天板と背面のプレートは樹脂製ですが，かなり質感が良いです．
特に天板，美しいです．

次にタッチパッドですね．
僕はよくタッチパッドを使う(マウスも使います)のですが，この感触も良いですね．
まぁASUSの方が好きですがVAIOの手触りも良いです．
そしてキーボード．
これはどこにも勝てないかな，ってところです．
絶妙な深さとクリック感でノートPCの中では1番気に入ってます．
以前使っていたSony VAIO SVT1112AJよりも静かなキーボードですが，それでも快適なキーボードです．

次はIOですかね…
IOは割とてんこ盛り仕様です．
  • Thunderbolt 4
  • Thunderbolt 4
  • USB 3.2 Gen 1x1 (Type A)
  • USB 3.2 Gen 1x1 (Type A)
  • HDMI (フルサイズ)
  • RJ45 Eathernet
  • 3.5 mm Mini-Jack

SDカードが刺さらないのは痛いですが，まぁ僕の手に届く範囲のLet's noteにも搭載されてないので悪しからず…というか端子構成は同じです．
給電・充電はThunderbolt 4経由でPD充電になります．
最大65 WなのでThe・ビジネスノートって感じです．(持ち運びの際に電源を小型化できるので助かる)

最後に驚いたところを書いて終わりにしようと思います．
このPCは指紋認証がすごいです．
ヒンジを開いた段階では電源は入りませんので(素晴らしい)，電源ボタンを押して起動する訳です．
その電源ボタンには指紋認証が内蔵されていて，そこがすごいのです．(語彙力…)
Windows Helloで指紋を登録して，VAIOハードウェア設定からある設定を有効にすると，起動時に電源ボタンに触れるその一瞬で認証が通ります．
とんでもないですよね．

おそらくですが，電源ボタンを押した段階ですぐに指紋のデータを取得します．
これはBIOSレベルで実行されます．
その後にWindowsが起動するのでLogonUI.exeが起動したら(ログオン画面が表示されたら)，ログオンプロセスに認証情報を渡します．
そうすることでログオンまで済んでしまうのです．
つまり登録した指で電源を入れれば10秒でデスクトップまで直行できます．
これはすげぇ… と思ったところでまたしばらく使って，感想を書こうと思います．

これからは研究も何もかもをこいつでやっていくので，大事に使っていきます．
ぜひご検討を．
