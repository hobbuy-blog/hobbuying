Title - YYYY.MM.DD

遂に…！ちゃんと新品のパソコンを購入しました！！
ということで今回の主役はVAIO F14 VJF1421です．
詳しいレビューは後日に書きますが，ここでは概略とファーストインプレッションだけ書いておきます．
新春セールの間に買ったので，ちょっと安く手に入りました．
参考程度に，価格はソフトケース(3,000円)込みで217,800円でした．破産するって．※高校生

ちなみに構成はガッツリ盛っています．
なにせ回路SIMソフト(LTspice)とかEDAソフト(KiCAD)とか3D CADソフト(Autodesk Fusion / Shapr3D)とかを同時起動しがちなので，RAMの量が欲しいわけです．
昨今の価格高騰はありますがここだけは外せない．
しっかり32 GB積んでいます．
あとはPro版Windowsも外せません．
そんなこんなで高級構成になってしまい，以下の構成です．

  • CPU : Intel Core 7 150U
    10C12T (2P8E) | 1.8/5.4 GHz
  • RAM : LPDDR5X 32 GB
  • ROM : PCIe 4.0 (NVMe) 1 TB
  • OS : Windows 11 Pro

なかなか盛り盛り構成です．
たぶんそこらの格安ゲーミングノートPCよりかは高いと思います．
ただ，めちゃくちゃ作り込みと言うか，品質と質感が素晴らしいんですよね．

初期セットアップを済ませたら，色々なソフトをインストールします．
代表例は以下のとおりです．
  • Autodesk Fusion Personal
  • Shapr3D
  • KiCAD
  • Visual Studio Code
  • Affinity

他にもありますがこれくらいにしておきます．
かれこれ全てのインストールに5時間くらいかかりました．
